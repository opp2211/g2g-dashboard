# g2g-dashboard

Дашборд для просмотра офферов G2G по валюте PoE с автообновлением.


## Скрипты

- `npm run dev` локальная разработка
- `npm run build` сборка
- `npm run preview` предпросмотр сборки
- `npm run lint` проверка ESLint

## Настройка

API URL и интервал обновления задаются в `src/constants/config.js`.
